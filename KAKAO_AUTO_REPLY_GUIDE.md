# 스케아트 카카오톡 채널 자동응답 설정 가이드

> 고객이 채널에 **먼저 메시지를 보내면** 자동으로 답장하는 기능이에요.
> **키워드형 자동응답**과 **챗봇(일반 상품)** 모두 **무료**입니다.
> 설정 위치: 카카오톡 채널 관리자센터 (PC: center-pf.kakao.com / 모바일: 카카오톡 채널 관리자 앱)

---

## 1. 가장 먼저 — 채팅 첫 화면 (웰컴 메시지)

고객이 채널 채팅방에 처음 들어왔을 때 뜨는 인사말이에요.

```
안녕하세요, 스케아트(skARTe)입니다 🎬
카메라·시네마 장비 대여 문의 감사합니다.

아래에서 원하시는 항목을 선택하시거나,
홈페이지에서 받으신 [접수번호]를 보내주세요.

▸ 대여 문의 / 예약 확인
▸ 추가 장비 요청
▸ 이용 안내 (요금·기간·반납)
▸ 오시는 길
```

---

## 2. 키워드형 자동응답 (무료, 설정 간단)

특정 단어가 오면 정해진 답장이 나가요. 아래 키워드별로 등록하세요.

### ① 접수번호 안내 — 키워드: `접수번호`, `#`, `예약`, `문의`

```
접수번호와 함께 아래 정보를 보내주시면 가장 빠르게 확인됩니다 🙏

1) 접수번호 (예: #1042)
2) 성함
3) 연락처

확인 후 담당자가 순서대로 안내해 드릴게요.
(영업시간: 평일 10:00~19:00)
```

### ② 대여 문의 — 키워드: `대여`, `렌탈`, `장비`, `견적`

```
대여 문의 감사합니다! 📷

홈페이지 장바구니에서 장비를 담고 "카카오톡 문의"를 누르시면
접수번호가 발급됩니다. 그 번호를 보내주시면 더 빠르게 진행돼요.

· 대여 기간(3일↑ 10%, 7일↑ 20% 할인)
· 수령 방식(방문/택배)
도 함께 알려주시면 정확한 견적을 드릴게요.
```

### ③ 추가 장비 요청 — 키워드: `추가장비`, `없는장비`, `수급`

```
보유 목록에 없는 장비도 수급 가능한 경우가 있어요!

필요하신 장비명, 사용 날짜, 수량을 보내주시면
보유·수급 여부를 확인해 안내해 드립니다.
```

### ④ 이용 안내 — 키워드: `이용`, `요금`, `기간`, `반납`, `가격`

```
[이용 안내] 💡

· 대여 단위: 1일 기준 (24시간)
· 할인: 3일 이상 10% / 7일 이상 20%
· 결제: 계좌이체 (예약 확정 시 안내)
· 수령/반납: 방문 또는 택배

자세한 내용은 홈페이지 '이용안내'를 참고해 주세요.
```

### ⑤ 위치/방문 — 키워드: `위치`, `오시는길`, `주소`, `방문`

```
[스케아트 오시는 길] 📍

인천 미추홀구 인하로 100
김현태인하드림센터 6층

· 방문 전 예약 필수
· 주차 가능 여부는 미리 문의 주세요
```

### ⑥ 영업시간 — 키워드: `영업`, `시간`, `운영`

```
[영업시간] ⏰
평일 10:00 ~ 19:00
주말·공휴일 휴무

영업시간 외 문의는 순차적으로 답변드려요.
남겨주시면 확인 후 연락드리겠습니다!
```

## 3. 챗봇(버튼형)으로 만들 경우 — 무료

키워드 대신 버튼을 눌러 진행하는 방식이에요. 흐름 예시:

```
[시작]
 └ "무엇을 도와드릴까요?"
     ├─ [대여 문의/예약확인] → 접수번호 입력 안내 → 상담원 연결
     ├─ [추가 장비 요청] → 장비명·날짜 입력 안내
     ├─ [이용 안내] → 요금/기간/반납 정보 카드
     ├─ [오시는 길] → 주소 + 지도 링크
     └─ [상담원 연결] → "잠시만요, 담당자가 확인 중이에요"
```

> 버튼형은 관리자센터 '챗봇' 메뉴에서 블록을 드래그로 구성해요.
> 키워드형보다 깔끔하지만 설정에 시간이 좀 더 걸립니다.

## 4. 운영 팁

- **부재중 메시지**: 영업시간 외엔 "확인 후 순차 답변" 자동응답을 걸어두면 고객이 기다림을 이해해요.
- **접수번호 매칭**: 고객이 보낸 접수번호(#1042)를 사이트 관리자 → "문의 관리" 탭에서 검색 → 수락하면 예약 일정에 자동 등록됩니다.
- **사칭 방지**: 채널 관리자센터에서 '비즈니스 채널' 인증을 받으면 인증 배지가 붙어요.
- **무료 범위**: 고객이 먼저 보낸 메시지에 대한 자동응답·챗봇 응답은 무료. 단, 사장님이 고객에게 *먼저* 보내는 단체 메시지는 월 1만 건까지만 무료(이후 건당 과금).

## 5. 설정 순서 요약

1. 카카오톡 채널 만들기 (없다면) → center-pf.kakao.com
2. 채널 관리자센터 → [채팅] → [자동응답] 또는 [챗봇]
3. 웰컴 메시지 등록 (위 1번)
4. 키워드별 자동응답 등록 (위 2번)
5. 홈페이지 KAKAO_URL을 실제 채널 채팅 URL로 교체
6. 테스트: 다른 계정으로 채널에 키워드 보내보기
